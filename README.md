## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/wankhade-Kunal/DevAssist-AI.git
cd DevAssist-AI
```

---

### 2️⃣ Backend Setup

```bash
cd server
npm install
```

Create `.env` file:

```
OPENAI_API_KEY=your_api_key_here
PORT=5000
```

Run backend:

```bash
npm run dev
```

---

### 3️⃣ Frontend Setup

```bash
cd client
npm install
npm start
```
